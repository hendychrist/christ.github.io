Kodekiddo Mobile App

Pada bagian “Login”, User diminta untuk memasukan Username dan password yang sudah terdaftar di dalam database. Bagi calon user yang belum mendaftar, dapat langsung mendaftar melalui tulisan “Sign Up” yang akan membawa user ke page “Register” untuk mengisi data diri yang dibutuhkan system.

Pada bagian “Register” ini User diminta untuk mengisi beberapa informasi seperti Full Name, Username, Password, dll untuk user yang belum mendaftar. Jika user sudah mengisi bagian Register maka user akan dialihkan ke page “Login”. Disini saya juga memberikan beberapa animasi seperti pada saat username atau password dalam kondisi gagal ataupun sukses.

User juga memiliki pilihan forgot password, yang dimana akan dikirimkan melalui email user yang sudah terdaftar. Setelah login user dapat melihat home dan feed yang ada pada aplikasi tersebut yang bertujuan untuk mengetahui update - update terbaru yang diadakan oleh Kodekiddo. Seperti lomba dan lainya.

Pada tap Account user juga dapat mengubah identitas profile mereka, seperti photo profile, username dan password. Serta info untuk mengetahui NIM dan achievment apa saja yang sudah mereka capai.

Pada tab About disini user dapat melihat beberapa TabBar. About untuk menunjukan tentang Kodekiddo. Lalu find us untuk menunjukan kontak cabang - cabang kodekiddo dari setiap kota. Contohnya seperti nomor Whatsapp dan titik detail lokasi melalui google maps. User dapat memilih melalui dropdown komponen yang ada pada screen tersebut.

lalu pada tab Contact Us digunakan untuk membuat suatu formulir untuk menghubungi pihak kodekiddo.
